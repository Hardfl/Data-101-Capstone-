# Spotify Popularity Analysis (Data 101 Capstone)

This project analyzes the Spotify Top 50 Songs of 2019 dataset using R to explore
whether musical features can explain and predict song popularity.

## Methods Used
- Exploratory Data Analysis (EDA)
- Hypothesis Testing & Permutation Tests
- Confidence Intervals & CLT
- Bayesian Updating
- Decision Trees (rpart)
- Association Rule Mining (Apriori)
- Multiple Hypothesis Testing Corrections

## Key Questions
- Are longer songs more popular?
- Do high-energy songs tend to rank higher?
- Can musical features predict popularity?

## Tools
- R
- rpart, arules, HypothesisTesting
- Base R visualization

## Results
The analysis demonstrates how data can both mislead and inform, highlighting the
importance of statistical reasoning when interpreting trends in popular music.
